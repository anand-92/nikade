### 问题

- **"lsls" 翻倍根因**：最终确认不是键盘处理 bug，而是 zsh-autosuggestions 颜色渲染错误（SGR 38 裸序列），根因是 terminfo 缺失
- **SGR 38 警告**：`io_handler: unimplemented or unknown SGR attribute: .{ .full = { 38 }, .partial = { 38 } }` — 可能来自 oh-my-zsh 主题发送的裸 `\e[38m]` 序列，terminfo 修复后频率降低但仍偶尔出现
