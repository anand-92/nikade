# REQ-003: 文件浏览器

## 概述

目录树浏览器，支持文件导航和 Git 状态着色。

## 核心需求

### P0 — 基础文件树

- [ ] 递归目录树展示，点击展开/折叠
- [ ] .gitignore 过滤（默认隐藏被忽略的文件）
- [ ] 目录优先 + 字母序排序
- [ ] 文件图标（按扩展名映射）

### P0 — Git 状态着色

- [ ] 绿色 = 新增，黄色 = 修改，红色 = 删除，蓝色 = 重命名
- [ ] 颜色从子文件递归传播到父目录

### P1 — 交互

- [ ] 右键菜单：在终端中打开、在 Finder 中显示、复制路径
- [ ] 点击变更文件 → 打开 Diff 视图
- [ ] 点击普通文件 → 只读预览（语法高亮）

### P2 — 增强

- [ ] 文件搜索（Cmd+P 快速查找）
- [ ] 拖拽文件到终端
